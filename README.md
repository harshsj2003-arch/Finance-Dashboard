# Finance Dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Features
- Data Cleaning
- Financial Data Analysis
- Month-wise Revenue Analysis
- Transaction Type Analysis
- Dashboard Creation

## Objective
Analyze financial transactions and create an interactive dashboard to track revenue, expenses, and business performance.

## Key Insights
- Monthly Financial Performance
- Accounts Payable Analysis
- Accounts Receivable Analysis
- Sales and COGS Analysis
- Data Visualization using Charts
